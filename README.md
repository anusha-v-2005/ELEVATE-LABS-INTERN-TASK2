# To-Do List App

A simple and elegant To-Do List application built with HTML, CSS, and JavaScript.

## Features

- **Add Tasks**: Enter a new task in the input field and click "Add Task" or press Enter
- **Mark as Complete**: Click on any task to mark it as completed (shows strikethrough)
- **Delete Tasks**: Click the "Delete" button to remove a task from the list
- **Responsive Design**: Works on desktop and mobile devices

## How to Run

1. Simply open `index.html` in any web browser
2. No server or installation required
3. The app works offline

## Usage

1. **Adding a task**: Type in the input box and click "Add Task" or press Enter
2. **Completing a task**: Click on the task text to toggle completion
3. **Deleting a task**: Click the red "Delete" button next to the task
